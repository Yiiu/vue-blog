# vue-blog

vue+vue-router+express+mongoose制作的简单博客程序（自用）

===

### 暂时尚未完成（慎用）

/logon 注册

/login 登陆

/admin/add 添加文章

/admin/article:id 修改文章

/admin/addtype 添加标签

ps:tag在添加文章时会自动添加，删除文章时加入tag没有别的文章的话会自动删除。


简单的个人博客程序，暂时完成了一下功能：

- [x] admin登录
- [x] 文章发布
- [x] 文章浏览
- [x] 文章修改删除
- [x] 文章批量删除
- [x] 查找文章
- [x] markdown
- [x] 代码高亮
- [x] 文章标签、分类

### 安装：
node4.x:

> npm install

> npm run build

> node --harmony_destructuring app.js


node6.x:

> npm install

> npm run build

> node app.js
